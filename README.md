# Real-Time-Web-Chat-Application
A fully functional real-time web chat application that enables users to communicate instantly through a modern and responsive interface. The application supports live message exchange without page refresh using real-time communication technology.

**Features**
- Real-time messaging.
- Instant message updates without page reload.
- Multiple user support.
- Responsive and user-friendly interface.
- Clean and structured project architecture.
- Fast communication using real-time technology.

**Installation Steps**

1️⃣ Clone the repository
  git clone https://github.com/your-username/your-repo-name.git

2️⃣ Navigate to the project folder
  cd your-repo-name

3️⃣ Install dependencies
  npm install

4️⃣ Start the server
  npm start

5️⃣ Open in browser
  http://localhost:3000
